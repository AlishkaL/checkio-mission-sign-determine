如何完成表达式？

唯一剩下的情况是当num == 0.您需要返回"zero" 。 

if num> 0:
    return "positive"
elif num < 0:
    return "negative"
else:
    return "zero"
那么，我的最终解决方案应该是怎样的呢？

你只需将代码封装成函数表达式。强烈建议对输入和输出数据使用类型提示。

以下是整个解决方案：

def determine_sign(num: int) -> str:

    if num> 0:
        return "positive"
    elif num < 0:
        return "negative"
    else:
        return "zero"

由于使用return 可以结束执行，因此不需要条件分支，还可以简化代码。对于简短的条件，可以在一行中编写表达式。 

def determine_sign(num: int) -> str:

    if num> 0: return "positive"
    if num < 0: return "negative"
    return "zero"

当然，抓住它

使用嵌套的三元运算符。 

def determine_sign(num: int) -> str:

    return "positive" if num> 0 else ("negative" if num < 0 else "zero")

这个解决方案有点棘手。表达式给出了不同的值（0、1、-1）。使用它作为列表的索引，可以得到正确的单词。 

def determine_sign(num: int) -> str:

    return ["zero", "positive", "negative"][(num> 0) - (num < 0)]
