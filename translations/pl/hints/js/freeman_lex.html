Nie mam pojęcia, jak zacząć rozwiązywać tę misję.

Proszę wpaść na jakiś pomysł 💡! Proszę użyć wyrażenia warunkowego if/else if/else, aby sprawdzić wszystkie warianty i zwrócić prawidłowy wynik w każdym przypadku.

Ok, od czego powinienem zacząć?

Istnieją trzy możliwe warianty, więc zbudujmy całe wyrażenie if/else if/else jako ramkę. 

if (...)
    ...;
else if (...)
    ...;
else
    ...;
Co robić dalej?

Właściwie nie ma to znaczenia, ale niech pierwszym przypadkiem będzie num> 0. Oczywiście w tym przypadku musi Pan zwrócić "positive". 

if (num> 0)
    return "positive";
else if (...)
    ...;
else
    ...;
Nadal potrzebuję podpowiedzi.

Proszę użyć tej samej logiki, aby utworzyć drugi przypadek (num < 0) dla liczb ujemnych ("negative"). 

if (num> 0)
    return "positive";
else if (num < 0)
    return "negative";
else
    ...;
Jak zakończyć wyrażenie?

Pozostaje tylko przypadek, gdy num == 0. Musi Pan zwrócić "zero". 

if (num> 0)
    return "positive";
else if (num < 0)
    return "negative";
else
    return "zero";
Jak więc powinno wyglądać moje ostateczne rozwiązanie?

Proszę po prostu opakować swój kod w wyrażenie funkcyjne. Używanie podpowiedzi typu dla danych wejściowych i wyjściowych jest wysoce zalecane.

Oto całe rozwiązanie:

function determineSign(num: number): string {

    if (num> 0)
        return "positive";
    else if (num < 0)
        return "negative";
    else
        return "zero";
}

Ponieważ użycie return kończy wykonywanie, nie potrzebują Państwo rozgałęzień warunkowych i mogą uprościć kod. W przypadku krótkich warunków można napisać wyrażenie w jednej linii. 

function determineSign(num: number): string {

    if (num> 0) return "positive";
    if (num < 0) return "negative";
    return "zero";
}

Jasne, proszę łapać!

Proszę użyć zagnieżdżonego operatora trójargumentowego. 

function determineSign(num: number): string {

    return num> 0? "positive": (num < 0? "negative": "zero");
}

Wyrażenie daje różne wartości (0, 1, 2). Proszę użyć go jako indeksu tablicy i uzyskać właściwe słowo. 

function determineSign(num: number): string {

    return ["negative", "zero", "positive"][(Math.sign(num)) + 1];
}
