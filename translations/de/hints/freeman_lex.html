Ok, womit soll ich anfangen?

Es gibt drei mögliche Varianten, also lassen Sie uns den gesamten Ausdruck von if/elif/else als Rahmen aufbauen. 

if ...:
    ...
elif ...:
    ...
else:
    ...
Was ist als nächstes zu tun?

Es ist eigentlich egal, aber lassen Sie den ersten Fall num> 0 sein. Offensichtlich müssen Sie in diesem Fall "positive" zurückgeben. 

if num> 0:
    return "positive"
elif ...:
    ...
else:
    ...
Ich brauche noch einen Hinweis.

Verwenden Sie dieselbe Logik, um den zweiten Fall (num < 0) für negative Zahlen ("negative") zu erstellen. 

if num> 0:
    return "positive"
elif num < 0:
    return "negative"
else:
    ...
Wie kann man den Ausdruck beenden?

Der einzige Fall, der übrig bleibt, ist num == 0. Sie müssen "zero" zurückgeben. 

if num> 0:
    return "positive"
elif num < 0:
    return "negative"
else:
    return "zero"
Wie sollte also meine endgültige Lösung aussehen?

Sie müssen nur Ihren Code in einen Funktionsausdruck verpacken. Die Verwendung von Typ-Hinweisen für Eingabe- und Ausgabedaten wird dringend empfohlen.

Hier ist die gesamte Lösung:

def determine_sign(num: int) -> str:

    if num> 0:
        return "positive"
    elif num < 0:
        return "negative"
    else:
        return "zero"

Da die Verwendung von return die Ausführung beendet, brauchen Sie keine bedingten Verzweigungen und können den Code vereinfachen. Für kurze Bedingungen können Sie den Ausdruck in eine Zeile schreiben. 

def determine_sign(num: int) -> str:

    if num> 0: return "positive"
    if num < 0: return "negative"
    return "zero"

Klar, schnapp sie dir!

Verwenden Sie verschachtelte ternäre Operatoren. 

def determine_sign(num: int) -> str:

    return "positive" if num> 0 else ("negative" if num < 0 else "zero")

Diese Lösung ist ein wenig knifflig. Die Ausdrücke verwenden unterschiedliche Werte (0, 1, -1). Verwenden Sie ihn als Index der Liste und erhalten Sie das richtige Wort. 

def determine_sign(num: int) -> str:

    return ["zero", "positive", "negative"][(num> 0) - (num < 0)]
