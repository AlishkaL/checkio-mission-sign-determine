So, how my final solution should look like?

You just need to and wrap your code into function expression. Using type hints for input and output data are highly recommended.

Here is the whole solution:

function determineSign(num: number): string {

    if (num > 0)
        return "positive";
    else if (num < 0)
        return "negative";
    else
        return "zero";
}

Since using return ends the execution you don't need conditional branches and may simplify the code. For short conditions you may write expression in one line. 

function determineSign(num: number): string {

    if (num > 0) return "positive";
    if (num < 0) return "negative";
    return "zero";
}

Use nested ternary operator. 

function determineSign(num: number): string {

    return num > 0 ? "positive" : (num < 0 ? "negative" : "zero");
}

Expression gives use different values (0, 1, 2). Use it as index of the array and get the right word. 

function determineSign(num: number): string {

    return ["negative", "zero", "positive"][(Math.sign(num)) + 1];
}
